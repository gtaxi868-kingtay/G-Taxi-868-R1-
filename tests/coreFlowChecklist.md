# G-Taxi Core Flow Checklist
## Must pass before every build

- [ ] App opens without crash
- [ ] Login with email succeeds
- [ ] Map shows real streets
- [ ] Destination search returns results
- [ ] Fare estimate calculates
- [ ] Ride request creates successfully
- [ ] Driver receives ride notification
- [ ] Driver accepts ride
- [ ] Rider sees driver moving on map
- [ ] Chat works during trip
- [ ] Driver completes trip
- [ ] Receipt shows correct fare
- [ ] Rating submits successfully
- [ ] Wallet balance updates correctly

## Secondary checks
- [ ] Grocery shows Coming Soon (not crash)
- [ ] Laundry shows Coming Soon (not crash)
- [ ] Profile opens without crash
- [ ] Subscription card shows Coming Soon (not crash)
- [ ] Settings opens without crash
