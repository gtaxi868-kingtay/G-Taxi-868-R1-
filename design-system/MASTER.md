# G-TAXI PRO MAX — MASTER DESIGN SYSTEM (VIBRANT FUTURISM)

## 💎 CORE CONCEPT: "VIBRANT FUTURISM"
The high-tech edge of the logo meets the soft, organic premium feel of modern lifestyle apps. It's "Friendly Cyberpunk" — powerful but accessible.

## 🎨 COLOR PALETTE
- **Primary (Electric Cyan)**: `#00FFFF` — From Logo. Used for active indicators and high-energy accents.
- **Secondary (Vibe Purple)**: `#7B61FF` — From Logo. Used for primary surfaces and brand depth.
- **Background (Evening Mist)**: `#0A0A1F` — A very dark, slightly purplish void.
- **Surface (Satin Glass)**: `rgba(255, 255, 255, 0.08)` — Soft, thick glass with deep diffusion.
- **Accents**: 
  - `Gradient`: `#7B61FF` -> `#00FFFF` (Primary Buttons).
  - `Glow`: Cyan radiance for focus states.

## 🖋️ TYPOGRAPHY
- **Heading**: Inter Bold (High letter-spacing: -0.02em).
- **Body**: Inter Regular.
- **Mood**: Modern, Clean, Upbeat.

## ✨ KEY ELEMENTS
1. **Super-Soft Cards**: Radius `32px`. Deep, soft shadows (`shadowOpacity: 0.3, shadowRadius: 20`).
2. **Floation**: Components feel like they are floating in a 3D gradient space.
3. **Logo Centricity**: Every screen should feel like it belongs to the "G-TAXI" mark. Use the Cyan/Purple gradient as the "DNA" of all interactive elements.
4. **No Scanlines**: Clear, pristine surfaces only.
